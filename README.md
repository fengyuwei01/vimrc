# vimrc
My vimrc

## INSTALL
```
curl https://raw.githubusercontent.com/fengzi01/vimrc/master/install.sh -L > install.sh && sh install.sh
```
## FIX IT

Compile YouCompleteMe plugin:
```
cd ~/.vim/bundle/YouCompleteMe
./install.py --go-completer --clang-completer
```
